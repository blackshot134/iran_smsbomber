# iran_smsbomber
A CRAZY SMS BOMBER built specifically for Iranian numbers! With 85+ active Iranian APIs, each triggered 10 times, that's 850+ SMS messages hitting the target phone in seconds!
____________________________
📦 Required Dependencies
Minimum Requirements:
Python 3.7 or higher
pip (Python package manager)
Internet connection
Windows/Linux/MacOS
____________________________
Required Python Packages:
requests
urllib3
___________________________
Install Required Packages:
pip install requests urllib3
# or for Python 3
pip3 install requests urllib3
# If you get permission errors on Linux/Mac:
sudo pip3 install requests urllib3
# On Windows (if pip not found):
python -m pip install requests urllib3
_____________________________
run:
python3 sms_bomber.py
or 
python sms_bomber.py
